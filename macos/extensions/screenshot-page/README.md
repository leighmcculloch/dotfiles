# Full Page Screenshot

A Chrome extension that captures a screenshot of the entire current web page, including content beyond the visible viewport. Click the extension icon and a PNG of the full page is saved.

## Usage

Click the extension icon in the toolbar. The page is captured and saved with a filename like:

```
Web Page example.com - 2026-05-09 14-30-45.png
```

## Save location

Chrome extensions can only write to the browser's Downloads folder. To have screenshots land on the Desktop automatically, set Chrome's download location to `~/Desktop` in `Settings → Downloads`.
