Workflow rules for this repo:
- Commits: create every commit with the /commit skill, never a raw git commit. Never add a Co-Authored-By: Claude or a Claude-Session trailer.
- Pull requests: create every PR with the /pr skill.
- Branches: create every new branch with the /branch skill. If the current branch is named claude/..., or its name is a random selection of words or characters that doesn't describe the work on it (or the work about to go on it), the first action in the workflow is to run /branch to rename it to something descriptive without the claude/ prefix.
